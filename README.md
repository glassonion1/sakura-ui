# Sakura UI
Sakura UI - UI components built with Tailwind CSS for React.

## Install
```
$ npm install @glassonion1/sakura-ui
```
or
```
$ yarn add @glassonion1/sakura-ui
```

## Standard Components
- H1-H6 Headings
- Link
- Card
- InfoCard
- Ol
- Ul
- Table
- Caption
- Thead
- Tbody
- Th
- Tr
- Td
- Code
- Pre

## Form components
- Button
- IconButton
- Radio
- RadioGroup
- Select
- Text
- Textarea
